## HW6 - 判断括号是否匹配
### 题目描述
括号包括 ’(’, ’)’, ’{’, ’}’, ’[’ 和’]’,()[]、({()[]})是合法的,而({)}、{[}是非法的
### 输入
第一行为测试样本个数n，接下来每行为一个仅包含括号的字符串
### 输出
1表示合法，0不合法
### 样例输入
~~~
4
()[]
({()[]})
({)}
{[}
~~~
## 样例输出
~~~
1
1
0
0
~~~
